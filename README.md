# Product Analytics UI - Evolution Timeline

Interactive product analysis interface showcasing feature evolution across 3 categories (Beauty, Electronics, Fashion) over 3 phases (Months 1-2, 3-4, 5-6).

## Features

- **3 Categories**: Beauty, Electronics, Fashion
- **3 Phases**: Progressive feature rollout timeline
- **9 Unique Views**: Each category x phase combination
- **Inter Tight Font**: Clean, modern typography
- **Mobile-First**: Responsive design
- **Real Images**: Unsplash stock photos

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Lucide React Icons

## Development

```bash
npm install
npm run dev
```

## Deployment

Built with Vite for optimized production builds.

```bash
npm run build
```
